# Домашняя работа №1: Game of Thrones

## Описание
Предсказание выживаемости персонажей вселенной «Песни Льда и Пламени» (Игра Престолов) на основе их характеристик.

## Задачи
- Первичный анализ данных и обработка пропусков.
- Кодирование категориальных признаков.
- Обучение модели классификации для предсказания гибели персонажа.
- Сохранение предсказаний в формате CSV.

## Использованные методы и библиотеки
- Pandas (анализ данных)
- Scikit-learn (моделирование)
- Классификация

# Домашняя работа №2: Linear Models

## Описание
Реализация и исследование линейных моделей машинного обучения, в частности логистической регрессии с регуляризацией.

## Задачи
- Реализация собственной логистической регрессии (Elastic Logistic Regression).
- Применение StandardScaler для нормализации данных.
- Оценка качества модели с помощью кросс-валидации (cross_val_score).
- Создание пайплайнов (make_pipeline) в Sklearn.

## Использованные методы и библиотеки
- NumPy
- Scikit-learn (StandardScaler, cross_val_score, make_pipeline)
- Оптимизация
- Регуляризация (ElasticNet)
